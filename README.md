# e-comm-frontend
Welcome to E-commerce Dashboard Frontend Repo.
Its live: https://e-comm-frontend.herokuapp.com/signup

## Features:
1. Signup + Login
2. Add a product to your e commerce website.
3. View all products.
4. Edit a product.
5. Delete a product.
